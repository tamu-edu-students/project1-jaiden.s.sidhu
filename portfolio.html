Technical Portfolio

A look at some of my technical projects.

Multi-Modal RAG Processor

Built and deployed a document processor for multi-modal data using Langchain, Google Gemma, and Docling. The tool boosts average document data coverage from 60% to 95% for Hewlett-Packard Enterprise's use cases.

OpenAI-based Chatbot

Developed a chatbot using Python and Langchain, integrated with OpenAI APIs. Implemented Fernet encryption to ensure secure communication for a Sleep Medicine website, which handles large user traffic.

Libmath Package in C++

Built a custom math library in C++ as a clone of standard libraries, utilizing CMake for build automation and Google Test for unit testing. This project improved my understanding of test-driven development.

iPhone Reselling Cost Analyzer

Developed a tool which uses web scraping and mathematical analysis to determine the average third party seller cost of different iPhone models to determine the best selling price for a customer.

Video Game Reviewing Website

Created a full-stack web application for reviewing video games using HTML, CSS, JavaScript, and a backend server. Implemented reviewing, commenting, profile following, and admin functionalities.
